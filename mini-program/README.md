# 人生进度管理局 - 微信小程序版

本目录是“人生进度管理局”原工程的 1:1 微信小程序原生开发版本复刻。

## 开发技术栈
- 微信小程序原生开发 (WXML / WXSS / JS)
- 适配微信小程序的极简治愈系设计系统
- 本地存储方案 (wx.getStorageSync / wx.setStorageSync)

## 目录结构
- `pages/`: 页面目录
  - `index/`: 人生总进度 (主页)
  - `countdown/`: 重要倒计时
  - `cycle/`: 年度/月度/今日进度 & 待办卡片
  - `about/`: 关于我们
- `utils/`: 工具类 (日期处理、本地存储、常量)
- `images/`: 图标 (需自行添加)

## 如何运行
1. 打开“微信开发者工具”。
2. 点击“导入”。
3. 选择此 `mini-program` 目录。
4. AppID 可使用测试号或您自己的 AppID。

## 注意事项
- **图标引用**: 为保持 UI 美观，代码中引用了 `images/` 目录下的图标。如果图标不存在，页面会显示空白占位符。建议您从 Lucide Icons 下载对应的 SVG/PNG 图标并放入 `images/` 目录。
- **背景纹理**: Web 版中使用的背景图片已保留引用，如需小程序内展示，确保域名已配置在微信白名单中。

## 核心功能 1:1 对标
- ✅ 人生总进度：支持三种显示模式（圆环、网格、光环）。
- ✅ 时间行箧：支持置顶、删除、分类标记。
- ✅ 时间周期：支持年、月、日进度切换，以及“今日三待办”。
- ✅ UI 视觉：完美复刻原版的 HSL 调色体系方案及极简布局。
